파이썬 **Tkinter**를 활용하여 제작한 데스크톱 기반 MBTI 성격 유형 검사 프로그램입니다.
직관적인 GUI 환경에서 12개의 질문을 통해 빠르게 성격 유형을 분석하고 해설을 제공합니다.

## ✨ Key Features

* **Dual Mode Support**:
    * 🖥️ **GUI Mode**: `tkinter` 기반의 윈도우 애플리케이션 (사용자 친화적)
    * ⌨️ **CLI Mode**: 터미널 기반의 텍스트 인터페이스 (빠른 테스트용)
* **Simple Logic**: 4가지 척도(EI, SN, TF, JP)를 기반으로 한 12문항 알고리즘
* **Standalone**: 외부 라이브러리 의존성 없음 (Standard Library 사용)
* **Executable**: `PyInstaller`를 통해 `.exe` 실행 파일로 변환 가능
